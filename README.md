## Core Expertise Areas

- **Cybersecurity & Penetration Testing**  
  Web application, network, and infrastructure vulnerability assessment, malware analysis, reverse engineering, and designing resilient systems against attacks.

- **Backend Development**  
  Scalable API design, microservices architecture, database optimization, distributed systems, message queues, and caching strategies.

- **System Architecture & Infrastructure**  
  High-availability systems, load balancing, advanced monitoring with Prometheus and Grafana, centralized logging with ELK stack.

- **Cryptography & Secure Protocols**

## Notable Projects

| Project | Description | Tech Stack |
|---------|-------------|------------|
| **Xenon Suite** | A modular penetration testing framework with automated vulnerability scanning, network mapping, and exploit suggestion engine. Designed for red team operations. | Python, Bash, Docker, Nmap |
| **CipherGuard** | A hybrid cryptography library implementing AES-256-GCM, RSA-4096, and ChaCha20-Poly1305 with a simple CLI and Python bindings. | C, Python, OpenSSL |
| **NetSpector** | Real-time network traffic analyzer with anomaly detection, DDoS pattern recognition, and a Grafana-based dashboard. | Go, React, WebSockets, Grafana, Prometheus |
| **AutoPwn** | Automated exploit framework that scans for known CVEs and executes tailored payloads. Plugin system for custom exploit development. | Rust, Python, Metasploit RPC |
| **WebForge** | High-performance web framework with built-in security headers, rate limiting, JWT authentication, and ORM. Inspired by Django and Laravel. | PHP, JavaScript, MySQL, Redis |
| **TeleSecure** | A Telegram bot for server monitoring, real-time security alerts, and remote command execution with 2FA and audit logging. | Python, Telethon, Docker, Prometheus |
| **HashCracker** | GPU-accelerated hash cracker supporting MD5, SHA-1, SHA-256, bcrypt, and scrypt. Uses rainbow tables and dictionary attacks. | C++, CUDA, Python |
| **SecureChat** | End-to-end encrypted messaging app with ephemeral messages, self-destruct timers, and zero-knowledge architecture. | Kotlin, Swift, Signal Protocol, Firebase |
